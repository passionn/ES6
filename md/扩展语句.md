### 扩展语句
扩展语句允许在需要多个参数或多个元素或多个变量的位置扩展表达式。

用于函数调用
```
myfunction(...iterableObj);
```
用户数组自变量
```
[...iterableObj,4,5,6]
```

例如：
```
var parts = ['shoulder', 'knees'];
var lyrics = ['head', ...parts, 'and', 'toes']; // ["head", "shoulders", "knees", "and", "toes"]
```

更好的额push 方法；
```
var arr1=[0,1,2];
var arr2=[3,4,5];
arr1.push(...arr2);
```
